LaserTrig

Description

This process triggers the laser via DO trigger line(s).

Syntax

LaserTrig( TrigFreq, PhaseDeg, Shots )

  • TrigFreq: A scalar between 0.01 to 2,000. Interval as 1e-10.

    Maximun trigger frequency (Hz).

  • PhaseDeg: A scalar between 0 to 359.9. Interval as 8.38e-8.

    Trigger the laser at a specific phase of main RF.

  • Shots: An integer between 1 to 255.

    Number of laser shots.

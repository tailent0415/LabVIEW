RFStart

Description

Start the main RF output with a specific frequency (Freq) and specific amplitude (Amp).

Syntax

RFStart( Freq, Amp, WfmType )

  • Freq: A scalar between 0 to 2,000,000.

    Time in second for opening the ion gate for ion quantitation estimation.

  • Amp: A scalar between 0 to PriRFMaxAmp.

    Time in second for opening the ion gate for ion quantitation estimation.

  • WfmType: An integer between 0 to 1.
    0 | Consine base
    1 | Digital waveform

    Determine waveform type of main RF.

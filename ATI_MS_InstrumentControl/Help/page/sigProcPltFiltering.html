PltFiltering

Description

This function returns the plot data which has been applied by a digital filter.

Syntax

PltFiltering( pltData, ktype, kwidth )

Parameters:

  • pltData: plot class (PriPlt|SecPlt)
    The parameter can be whole plot data.
  • ktype: string.
    It defines the type of kernel used in the discrete convolution with the ydata.
    It can accept the following types of kernels.
    • norm( Gaussian function )
    • Gaussian( Gaussian function )
    • boxcar( Rectangular )
    • triang( Triangle )
    https://docs.scipy.org/doc/scipy/reference/signal.windows.html
  • kwidth: float.
    Width of the kernel in unit of second.

Return:

  • PltFiltering: plot class
    It returns filtered whole plot data.

UpdAuxArbPulses

Description

This process updates a look-up table which is used for auxiliary arbitrary waveform.
e.g. UpdAuxArbPulses( [0.001,0.500,0.750], [0.005,0.502,0.758], [1,0.5,-0.25] )
The above script writes three pulses at cycles of [0.001, 0.005], [0.500, 0.502], and [0.750, 0.758] with
intensity of 1, 0.5, and -0.25 into the look-up table of auxiliary arbitrary wave.

Syntax

UpdAuxArbPulses( CycleIni, CycleEnd, PulseInt )

  • CycleIni: A scalar array with 1 to 255 elements the element between 0 to 1. Interval as 6e-5.

    Expresses the initial cycles of pulses.

  • CycleEnd: A scalar array with 1 to 255 elements the element between 0 to 1. Interval as 6e-5.

    Expresses the end cycles of pulses.

  • PulseInt: A scalar array with 1 to 255 elements the element between -1 to 1. Interval as 7.5e-3.

    Presents the intensity of pulses.

Wait

Description

This process wait for a trigger from the DI.

Syntax

Wait( Timeout, DI, Edge )

  • Timeout: A scalar between 0.001 to 60000. Interval as 25Î¼s.

    The time period for idle in a unit of millisecond.

  • DI: An integer between 0 to 7.
  • Edge: An integer between 0 to 3.
    0 | timeout
    1 | rising
    2 | falling
    3 | both

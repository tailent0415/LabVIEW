SetDAQ

Description

This process sets the channel of data acquicision and its sampling rate.

Syntax

SetDAQ( Channel, SampRate )

  • Channel: An integer between 0 to 3.
    0 | do not acquisition data.
    1 | enquiry signal from AI0, the sampling rate is fixed to 200kHz.
    2 | enquiry signal from AI1, the sampling rate is fixed to 200kHz.
    3 | enquiry signal from pAI (Programmable AI), the user can set the sampling rate.
  • SampRate: An integer between 0 to 5.
    0 | 306.25kHz
    1 | 612.5
    2 | 1.25MHz
    3 | 2.5MHz
    4 | 5MHz
    5 | 10MHz

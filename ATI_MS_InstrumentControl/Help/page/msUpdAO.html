UpdAO

Description

This process updates the definitions of four AO channels (AO0:3).
If the function of the channel sets to 0, the output shall be updated to the user defined value, immediately.

Syntax

UpdAO( Func, Val )

  • Func: A integer array with 4 elements the element between 0 to 1.
    0 | User defined
    1 | connects to ion gate analogue, it is defined by process IonGate and UpdIGVolt.

    It defines the functions of AO0 to AO3.

  • Val: A scalar array with 4 elements the element between -10 to 10. Interval as 7.5e-9.

    It defines the user-defined values.

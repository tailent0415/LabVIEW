UpdIGVolts

Description

This process updates the analogue output of ion gate. The value is between -10 to 10.

Syntax

UpdIGVolts( OnVolt, OffVolt )

  • OnVolt: A scalar between -10 to 10.

    The voltage output via an AO channel at ON state.

  • OffVolt: A scalar between -10 to 10.

    The voltage output via an AO channel at OFF state.

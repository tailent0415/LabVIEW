UpdDO

Description

This process updates the definitions of DO channels (DO0:3).
If the function of the channel sets to 0, the output shall be updated to the user defined value, immediately.

Syntax

UpdDO( Func, Val )

  • Func: An integer array with 4 elements the element between 0 to 9.
    0 | User defined
    1 | Laser trigger
    2 | Ion gate on?
    3 | Scan?
    4 | Wait for cmd?
    5 | Not conj?
    6 | 1st conj?
    7 | No defined
    8 | No defined
    9 | No defined

    It define the functions of DO0 to DO3.

  • Val: A boolean array with 4 elements.

    The array used as user-defined values for DO.

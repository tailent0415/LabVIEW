PltConv1D

Description

It returns the plot data with the one-dimensional convolution of its ydata and a kernel via graphic processing unit. The NVIDIA CUDA is definitely required.

Syntax

PltConv1D( kernel, pltData )

Parameters:

  • kernel: float 1-D array
    It is a sequence of number with a given width and value. See MakeKernel for having further detail.
  • pltData: float 1-D array
    The parameter can be whole plot data.

Return:

  • PltConv1D: plot class
    It returns whole plot data with one-dimensional convolution of its ydata and the kernel.

Conv1D

Description

Returns the discrete, linear convolution of two one-dimensional sequences via graphic processing unit. The NVIDIA CUDA is definitely required.

Syntax

Conv1D( kernel, ydata )

Parameters:

  • kernel: float 1-D array
    It is a sequence of number with a given width and value. See MakeKernel for having further detail.
  • ydata: float 1-D array
    A sequence of discrete signal points.

Return:

  • Conv1D: plot class
    It returns whole plot data with one-dimensional convolution of its ydata and the kernel.

UpdRFChBias

Description

This process updates the biases of each RF channels.

Syntax

UpdRFChBias( Ch1, Ch2, Ch3 )

  • Ch1: A scalar between -1 to 1. Interval as 1e-4.

    It define the output bias of the RF 1.

  • Ch2: A scalar between -1 to 1. Interval as 1e-4.

    It define the output bias of the RF 2.

  • Ch3: A scalar between -1 to 1. Interval as 1e-4.

    It define the output bias of the RF 3.

Filtering

Description

This function filters the data along one dimension with a kernel.

Syntax

Filtering( dt, ydata, ktype, kwidth )

Parameters:

  • dt: float.
    Sampling time interval of signal in unit of second
  • ydata: float 1d-array.
    A sequence of discrete signal points
  • ktype: string.
    It defines the type of kernel used in the discrete convolution with the ydata.
    It can accept the following types of kernels.
    • norm( Gaussian function )
    • Gaussian( Gaussian function )
    • boxcar( Rectangular )
    • triang( Triangle )
    https://docs.scipy.org/doc/scipy/reference/signal.windows.html
  • kwidth: float.
    Width of the kernel in unit of second.

Return:

  • Filtering: float 1d-array
    The output of filter.

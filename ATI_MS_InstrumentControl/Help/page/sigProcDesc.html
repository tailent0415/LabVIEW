Data access

User can access the acquired scan signals via a class named PriPlt.ScanSig.

Variable Type Description
PriPlt.ScanSig[n].dt Float64 (DBL) The sampling time interval of data acquisition during the nth scan process.
PriPlt.ScanSig[n].ydata List of Float32 (SGL) The array listing acquired signal during the nth scan process.
PriPlt.ScanSig[n].xdata List of Float64 (DBL) The array listing m/z for ydata.

The ScanSig shall be displayed by a tGraph (graph display in time-domain) or an mzGraph (graph display in m/z-domain, mass spectra)
which can be selected by a checkbox on the main screen.

The label of graph displays can be modified via changing the values of below variables

Variable name Type Description
PriPlt.tGraph.xlabel String The label at x-axial of tGraph
PriPlt.tGraph.ylabel String The label at y-axial of tGraph
PriPlt.mzGraph.xlabel String The label at x-axial of mzGraph
PriPlt.mzGraph.ylabel String The label at y-axial of mzGraph

User can access the signal acquired during ion injection processes via a class named SecPlt.IonSig.

Variable name Type Description
SecPlt.IonSig[n].dt Float64 (DBL) The sampling time interval of data acquisition during the nth ion injection process.
SecPlt.IonSig[n].ydata List of Float32 (SGL) The array listing acquired signal during the nth ion injection process.

Its graph label can be modified via changing

Variable name Type Description
SecPlt.ionGraph.xlabel String The label at x-axial of ionGraph.
SecPlt.ionGraph.ylabel String The label at y-axial of ionGraph.

NumRectifier

Description

This function reconstructs the signal acquired by the CSPD 1st stage output. It returns the approached charge number in a while of sampling time interval.

Syntax

NumRectifier( dt, ydata )

Parameters:

  • dt: float.
    Sampling time interval of signal in unit of second
  • ydata: float 1d-array.
    A sequence of discrete signal points

Return:

  • NumRectifier: float 1d-array.
    The result executing via the numerical rectifier of CSPD

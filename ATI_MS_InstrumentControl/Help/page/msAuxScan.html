AuxScan

Description

This process scans the auxiliary waveform stepwisely.

Syntax

AuxScan( AuxWfmType, AuxStart, iAuxCyc, AuxCyc, Beta, BetaScale, AuxAmp, AuxPulse1, AuxPulse2, AuxPulseAmp, Time )

  • AuxWfmType: An integer between 0 to 3.
    0 | Cosine based
    1 | Square wave
    2 | Double pulses for each cycle
    3 | Arb wave, described by a process named UpdAuxArbPulses

    It determines the type of the auxilary waveform.

  • AuxStart: An integer between 0 to 127.

    The auxiliary waveform starts after a number of the RF conjunction for a step.

  • iAuxCyc: A scalar between 0 to 1. Interval as 2.4e-10.

    The initial phase of auxiliary AC.

  • AuxCyc: A scalar array with 2 elements the element between 0 to 99999. Interval as 2.4e-10.

    The number of active cycles and the total cycle of the auxiliary AC for a step.

  • Beta: A scalar array with 2 elements the element between 0.01 to 1.

    The Beta value is used to define the auxiliary AC frequency and m/z scale of the acquired mass spectra. The array denotes the indicating the initial Beta and final Beta of a scan.

  • BetaScale: A scalar between 0 to 10.

    It defines the actual ratio (R) between auxiliary frequency and main RF frequency by R = Beta * BetaScale / 2.

  • AuxAmp: A scalar array with 2 elements the element between 0 to 1.

    The array denotes the indicates the intial auxiliary AC amplitude and final auxiliary AC amplitude for waveform types of cosine wave and square wave.

  • AuxPulse1: A scalar array with 2 elements the element between 0 to 1. Interval as 2.4e-10.

    The array denotes the beginning and ending auxiliary AC cycles of the first pulse.

  • AuxPulse2: A scalar array with 2 elements the element between 0 to 1. Interval as 2.4e-10.

    The array denotes the beginning and ending auxiliary AC cycles of the second pulse.

  • AuxPulseAmp: A scalar array with 2 elements the element between -1 to 1.

    The array denotes the amplitudes of AuxPulses.

  • Time: A scalar between 0 to 60.

    Scan time in unit of second.

MakeKernel

Description

This function returns a window (kernel) with given type and width.

Syntax

MakeKernel( dt, ktype, kwidth )

Parameters:

  • dt: float.
    Sampling time interval of signal in unit of second
  • ktype: string.
    It defines the type of kernel used in the discrete convolution with the ydata.
    It can accept the following types of kernels.
    • norm( Gaussian function )
    • Gaussian( Gaussian function )
    • boxcar( Rectangular )
    • triang( Triangle )
  • kwidth: float.
    Width of the kernel in unit of second.

Return:

  • MakeKernel: float 1d-array
    A one-dimension array with specific length and shape.

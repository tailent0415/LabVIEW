UpdConj

Description

This process updates the conjunctions which produce tiny pauses at specific RF phases.
Moreover, they are the phase flags of main RF used in the scan waveform calculations.

Syntax

UpdConj( IniPhase, ConjLen, PhaseDiff, FixedLen )

  • IniPhase: An integer between 0 to 4294967295.

    The initial phase of a conjunction.

  • ConjLen: A scalar array with 1 to 16 elements the element between 0 to 49.999. Interval as 1e-3.

    The length of conjunction. It expresses as the ratio (in percentage) to wavelength or fixed time interval (in second) as a conjunction.

  • PhaseDiff: An integer array where the size is equal to the ConjLen array the element between 0 to 4294967295.

    The phase difference between conjunctions.

  • FixedLen: A Boolean value.
    T | The length of conjunction is fixed.
    F | The length of conjunction is a ratio of RF period.

    Determine that the length of conjunction.

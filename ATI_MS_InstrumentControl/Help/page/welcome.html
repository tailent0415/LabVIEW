For supporting Python syntax in the script edition of MS processes and signal post-processing. Python 3.7.4 is certainly required for your PC.
Python 3.7.4 Download

Python installation


  • Tick the checkbox: Add Python To Path
  • Click Install Now for installation

Installation of libraries

  • Execute cmd.exe
  • Fill the command line and press enter for each as below:
    PATH is the installation path.
    pip install "PATH\System\numpy-1.17.1-cp37-cp37m-win_amd64.whl"
    pip install "PATH\System\scipy-1.3.1-cp37-cp37m-win_amd64.whl"
    pip install "PATH\System\numba-0.46.0-cp37-cp37m-win_amd64.whl"
